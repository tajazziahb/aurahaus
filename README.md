##  How It Works
- Users can **sign up** and **log in** to manage their account  
- The **dashboard** displays account details, project statistics, and summaries  
- Users can **add new projects** with:
  - Title  
  - Description  
  - Type  
  - Status  
  - Start and end dates  
  - Cost  
- Summary section dynamically updates total values and status counts
